ROMFoam: Reduced-order Modeling with OpenFOAM
---------------------------------------------

ROMFoam contains libraries and executives to run reduced-order modeling (ROM) with OpenFOAM. ROMFoam uses libraries from `DAFoam <https://github.com/mdolab/dafoam>`_.

The ROMFoam repository comprises of five main directories, and the source code is available on `GitHub <https://github.com/mdolab/romfoam>`_.

- applications: ROM solvers and utilities
- doc: documentation
- python: python interface to other optimization packages
- src: the core DAFoam and ROMFoam libraries
- tutorials: sample optimization setup for each ROM solver
